## Marketing Agent

AI-powered marketplace listing optimizer that transforms weak project descriptions into structured, conversion-ready listings.

## Problem Statement

Marketplace listings often underperform due to weak copy, inconsistent metadata, and unclear visuals. As a result, high-quality projects fail to gain traction, leading to lower clicks, engagement, and conversions.

##  Project Goal

Build an AI-driven marketing agent that transforms raw project descriptions into stronger, structured marketplace listings by generating:

- Clear short and long descriptions

- 5 value-driven bullet highlights

- Clean, structured metadata

- A marketing image plan

- A guardrail validation report to prevent overpromising
